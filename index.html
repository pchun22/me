Some of my work

Projects

  • BTS MV Detector

    A web application that detects what BTS music video an uploaded image is from

    Motivation: The music videos for BTS's (the arguably most popular K-pop group) songs are very artistic and often have a visual theme that make them so memorable. I wanted to see if these themes were noticeable and identifiable to a computer through machine learning

    What: I took the top 30 BTS music videos from YouTube and ran them through a Python script to take a frame every 5 seconds. I then used these images to train a image classification model with Microsoft Azure's customvision.ai and then developed a basic front-end to call the classification model in the back-end and return the song name.

    Where: Here!

  • Event Bot Sync

    An automation bot that syncs events from an external calendar to Google calendar

    Motivation: In my sophomore internship, we had a separate calendar for our intern events options. I discovered that some interns were unaware these event opportunities because they only were checking google calendar, and wanted to build something that would sync the two calendars together

    What: I used an internal automation tool that I heard about in a speaker panel. By learning how to use this tool in a couple days, I built a process that scrapes and structures the event data from the intern portal and then uploads that data into google calendar to create the events in a single calendar.

    Where: Internal company app marketplace

  • Trace Dance

    A web portal to show you and the original dance choreography side by side

    Motivation: When learning new dances, I saw a pain point in being able to assess myself against the original choreo. I talked to other dancers and learned their solution was to put a recording of themselves next to a video of the original choreography in iMovie. This takes a long time so I set out to find a better way

    What: I embedded a webcam and a YouTube iframe into a webpage to visualize myself and a choreography side by side in real time. I'm currently working on recording the videos and playing them back together. There is potential to use computer vision to trace both dancers' outlines and create an accuracy score.

    Where: WIP - Stay tuned!
