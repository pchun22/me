Projects

BTS MV Detector

A web application that detects what BTS music video an uploaded image is from

Motivation: The music videos for BTS's songs often have an artistic visual theme. I wanted to see if these themes were noticeable and identifiable to a computer through machine learning.

What: I took the top 30 BTS music videos from YouTube and ran them through a Python script to take a frame every 5 seconds. I then used these images to train a image classification model that is callable from a web front-end which is accessible here.

Event Bot Sync

An automation bot that syncs events from an external calendar to Google Calendar

Motivation: In a summer internship, I discovered interns were missing event opportunities due to not checking their intern specific calendar. I decided to build something that would help them see those events.

What: I learned an automation tool that I heard about in a speaker panel and used it to build an automated process that scrapes and structures the intern calendar portal and then uploads that data into Google Calendar. This tool is now uploaded into the firms internal app marketplace.

Trace Dance

A web portal to show you and the original dance choreography side by side

Motivation: When learning new dances, I saw a pain point in being able to assess myself against the original choreo. I spoke to other dancers and learned their current solution was quite manual so am working to find a better way.

What: I embedded a webcam and YouTube frame into a webpage to show myself and a choreography side by side in real time. I'm working on recording for synced playback and using computer vision to create an accuracy score.
